Proto-GDC is a low-sample relation extraction model calibrated using Gaussian distribution. It generates pseudo instances by querying several nearest neighbors of the support set instances to calibrate the prototype bias.

The Proto-GDC method is based on the followings papers：
“FewRel: A Large-Scale Supervised Few-Sample Relation Classification Dataset with State-of-the-Art Evaluation Results” 
“Free Lunch for Few-shot Learning: Distribution Calibration”
“Boosting Few-Shot Text Classification via Distribution Estimation”


The Proto-GDC codes and dataset is based on the following: 

https://github.com/thunlp/FewRel

https://github.com/ShuoYang-1998/Few_Shot_Distribution_Calibration
